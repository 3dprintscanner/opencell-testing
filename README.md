## How To Run

 - Ruby Version `2.4.2`
 - Requires running Postgres db with user called `postgres` and password of `password` on `localhost:5432`
 - Requires`node.js`
 - `bundle install`
 - `npm install`
 - `rake db:create`
 - `rake db:schema:load`
 - `rake db:seed`
 - `rails s`
 - an admin dashboard is available at `/admin` with username: `admin@example.com` and password `password`
